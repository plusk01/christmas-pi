Christmas Pi
============

Raspberry Pi controlled Christmas Lights that play to MIDI


RaspberryPi Setup
-----------------

1. [Install NOOBS](http://www.raspberrypi.org/downloads/)
2. Using NOOBS, install Raspbian.
3. Using Rasp-Config, setup terminal booting only.
4. Configure wifi dongle by: [See](http://thepihut.com/blogs/raspberry-pi-tutorials/16018016-how-to-setup-wifi-on-the-raspberry-pi-raspbian)
